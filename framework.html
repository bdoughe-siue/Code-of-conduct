PMI Ethical Decision-Making Framework

A systematic approach to making ethical decisions aligned with PMI's Code of Ethics

1
Assessment
Identify and evaluate the ethical issue at hand, considering all perspectives and stakeholders involved.
Click to see key questions →
STEP 1
Assessment
Key Questions
What is the ethical issue or dilemma?
Who are the stakeholders affected?
What are the facts of the situation?
What are my initial thoughts and feelings?
Is there an apparent conflict between ethical principles?
What PMI Code of Ethics principles apply here?
2
Alternatives
Explore multiple courses of action and their potential consequences on all stakeholders.
Click to see key questions →
STEP 2
Alternatives
Key Questions
What are all possible courses of action?
What are the consequences of each alternative?
How does each option align with PMI values?
Which stakeholders benefit or suffer from each option?
Are there creative solutions that satisfy multiple principles?
What would respected colleagues suggest?
3
Analysis
Evaluate each alternative against ethical principles, considering both aspirational and mandatory standards.
Click to see key questions →
STEP 3
Analysis
Key Questions
Which option best upholds Responsibility?
Which option demonstrates Respect for all parties?
Which option ensures Fairness?
Which option exemplifies Honesty?
Does this meet mandatory requirements?
Does this aspire to the highest standards?
4
Application
Make a decision, implement it with integrity, and reflect on the outcome for continuous improvement.
Click to see key questions →
STEP 4
Application
Key Questions
What is my decision and rationale?
Can I defend this decision publicly?
How will I implement this decision?
Who needs to be informed and when?
What did I learn from this situation?
How can I prevent similar dilemmas in the future?

Remember: When in doubt, consult with trusted advisors, refer to PMI's Code of Ethics, and choose the path that upholds the highest ethical standards.
